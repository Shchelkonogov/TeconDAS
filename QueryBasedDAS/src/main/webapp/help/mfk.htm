Подсистема сбора данных с контроллеров МФК-1500 осуществляет сбор информации с контроллеров МФК-1500, установленных на объектах диспетчеризации, и передачу этой информации по структурам БД АС «ТЕКОН-Диспетчеризация».
Интерфейс подсистемы сбора с MFK-1500
Подсистема сбора данных с контроллеров МФК-1500 блокирует объекты, превышающие объём передачи суточного трафика, до истечения суток. В таблице имя таких объектов отображается красным (как на примере ниже).
Блокировка объектов свидетельствует о прекращении информационного обмена между контроллером МФК-1500 и АССД. Для возобновления обмена можно выполнить действие

Значения параметров в колонке «Имя» у окна «Системные параметры»:
Время
 
Время прибора на объекте
 
ARC_WRITE_HOURS
 
Время, за которое один раз в указанное количество часов архивируются поступившие данные.
 
TS_READ_DELAY
 
При совместной передаче данных приборов автоматики и данных теплосчётчика на объекте, параметр определяет время, через которое Система будет запрашивать данные при отсутствии их автоматического поступления.
Время указано в миллисекундах.
 
ARC_WRITE_MINUTES
 
Время, за которое один раз в указанное количество минут архивируются поступившие данные.
 
Рассинхронизация времени
 
Несоответствие времени прибора на объекте и времени Системы.
