Автоматизированная система сбора данных

«ТЕКОН»
Графический интерфейс пользователя Автоматизированной системы сбора данных (АССД) «ТЕКОН» предназначен для отслеживания получения данных от приборов, установленных на объектах диспетчеризации.
Интерфейс позволяет:
  •      проверять соответствие полученных параметров с системы сбора и параметров, зарегистрированных в АС «ТЕКОН-Диспетчеризация»,
  •      обнаруживать неисправности приборов учёта,
  •      просматривать архивы передачи групп данных с определённой периодичностью,
  •      оперативно анализировать передачу данных объектами, непосредственно от приборов до АС «ТЕКОН-Диспетчеризация»,
  •      контролировать информационный трафик для каждого объекта в рамках выделенного ежесуточного и ежемесячного лимитов,
  •      формировать отчётность по подключенным объектам.
Переход в интерфейс АССД «ТЕКОН» осуществляется из формы «Линковщик» в АС «ТЕКОН-Диспетчеризация». Вход в интерфейс выполняется при введении логина и пароля, полученных у Администратора.
